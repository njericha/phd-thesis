# phd-thesis
Collection of code and data for my PhD thesis at The University of British Columbia in Mathematics.

## To Run
Clone or download the repo, and then in the Julia REPL, `] activate .` the project before running `] instantiate` to download all the required packages.

You will need to unzip `transcriptomics_data.zip` (~33 MB uncompressed) to be able to run the transcriptomics.
